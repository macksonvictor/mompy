## 💚 O Que O Mompy Faz

Mompy transforma a prática de programação em um fluxo de aprendizado focado. Em vez de começar com um editor vazio, ele entrega uma experiência de console de treinamento retrô com:

- aulas guiadas antes de cada bloco de missões;
- missões organizadas por dificuldade real;
- validação em Python no backend;
- execução do código do estudante em processo separado;
- progresso, XP e nível salvos localmente;
- interface CRT/industrial onde Mompy reage a ações, erros e acertos.

A camada visual usa HTML, CSS e JavaScript. A lógica do aplicativo, dados das missões, validação, progresso, XP e shell desktop são feitos em Python.

---

## 🧠 Por Que Existe

Muitas ferramentas para iniciantes pulam rápido demais para conceitos como `if`, loops e funções. Mompy foi pensado para ensinar com uma curva mais suave:

1. Mostrar texto.
2. Guardar valores.
3. Tomar decisões.
4. Repetir ações.
5. Trabalhar com vários valores.
6. Organizar código com funções.

Cada missão deve pedir apenas conceitos que já foram apresentados em uma aula guiada anterior.

## 🐍 Caminho De Aprendizado

| Bloco | Missões | Tema | Conceitos |
| --- | --- | --- | --- |
| 01 | 01-05 | Primeiros comandos | `print`, strings, aspas, parênteses, saída no terminal |
| 02 | 06-10 | Variáveis e valores | nomes de variáveis, atribuição com `=`, strings, números |
| 03 | 11-15 | Decisões | booleanos, `if`, `else`, comparações, `==`, indentação |
| 04 | 16-20 | Repetição | `for`, `range`, variável do loop, indentação |
| 05 | 21-25 | Listas | colchetes, itens, vírgulas, índices, iteração |
| 06 | 26-30 | Funções | `def`, chamadas, parâmetros, `return`, indentação |

## 🚀 Executar Pelo Código-Fonte

Instale as dependências:

```bash
python -m pip install -r requirements.txt
```

Abra como app desktop:

```bash
python main.py
```

Verifique o backend sem abrir janela:

```bash
python main.py --check
```
